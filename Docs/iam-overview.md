# IAM Overview

Identity and Access Management (IAM) in OCI is responsible for controlling access to cloud resources.

IAM manages:

- Authentication (verifying identity)
- Authorization (granting permissions)

OCI IAM uses a policy-based access control model to ensure secure resource access.

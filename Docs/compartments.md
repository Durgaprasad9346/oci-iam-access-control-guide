# Compartments in OCI

Compartments are logical containers used to organize and isolate cloud resources.

They allow administrators to structure resources and apply access control boundaries.

---

## Example Compartment Structure

Root Tenancy
│
├── Development
│ ├── Compute
│ └── Storage
│
└── Production
├── Application Servers
└── Databases


---

## Benefits of Compartments

- Resource isolation
- Access control boundaries
- Organizational structure for cloud resources
